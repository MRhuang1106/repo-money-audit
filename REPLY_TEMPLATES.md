# Reply Templates

Use these after someone responds to an outreach issue.

## If They Say "PR Welcome"

```markdown
Thanks. I can open a small PR focused only on README structure and repo trust signals.

I'll keep it scoped:

- clearer quick start
- one usage example
- issue/funding/contribution suggestions if relevant

I'll avoid changing code behavior.
```

## If They Say "Thanks"

```markdown
Glad it helped. If you want, I can draft a small README outline in this issue so you can decide whether it is worth turning into a PR.
```

## If They Ask For Help

```markdown
Yes. I can help in two ways:

1. Free: a short README outline in this issue.
2. Paid: a complete README/repo cleanup with install steps, examples, templates, and support links.

My paid service details are here:
https://github.com/MRhuang1106/repo-money-audit/blob/main/SERVICES.md
```

## If They Reject

```markdown
No problem. Thanks for taking a look, and good luck with the project.
```

## If They Do Not Reply

Wait 3-5 days. Do not bump the issue repeatedly.

Move to a new lead instead.
